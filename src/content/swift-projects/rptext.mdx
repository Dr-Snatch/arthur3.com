RPtext is a native macOS app built with SwiftUI that uses LLMs to generate an entire text RPG in real time. The AI handles narrative, dialogue, combat descriptions, and world reactions — all with no scripted content.

The app supports both Claude API and local Ollama models. Responses stream token-by-token to the UI using Swift's async/await and AsyncSequence, while structured JSON game state updates are buffered and parsed separately.

The "XML sandwich" prompt architecture ensures reliable structured output even from smaller local models, with a multi-fallback parsing chain that keeps the game running even when the model produces malformed JSON.
