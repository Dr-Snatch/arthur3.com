BeatMap is a native iOS app built entirely in SwiftUI. It lets you create diary entries linked to songs, locations, and personal metrics like mood, nostalgia, and energy.

The app integrates with Spotify for song search and playback previews, uses ShazamKit for song identification via the microphone, and pins entries on a MapKit map. All data is persisted with Core Data after migrating from UserDefaults in v1.1.0.

The OAuth 2.0 PKCE flow for Spotify was built from scratch — code verifier generation, SHA256 with CryptoKit, Keychain token storage, and silent background refresh.
