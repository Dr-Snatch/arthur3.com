## What it is

A personal dashboard that pulls together everything I need to see in one place: recent GitHub commits, open PRs, a task list, and quick notes. Built with React and TypeScript, deployed on Cloudflare Workers.

## Why I built it

I was switching between GitHub, Notion, and my terminal constantly to keep track of what I was working on. Devboard puts all of that in a single tab that loads in under a second.

## Technical details

- **GitHub integration** — fetches recent commits, PRs, and repo activity via the GitHub API, cached at the edge
- **Task management** — simple CRUD with local persistence, no external database needed
- **Quick notes** — markdown-enabled scratchpad for thoughts that don't deserve a full document
- **Cloudflare Workers** — server-side API routes that handle GitHub auth and caching, keeping the client lightweight

## What I learned

This was my first proper TypeScript/React project. Coming from Swift, the type system felt familiar but the ecosystem was overwhelming — the number of ways to do anything in JavaScript is staggering. I learned to pick a pattern and commit to it rather than chasing the "best" approach.
